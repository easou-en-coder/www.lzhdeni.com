About Get the Framework

Start here: build a productivity system that actually works

This guide shows how to navigate the Lzhdeni knowledge system and assemble practical frameworks for remote work.

How to use this site

Lzhdeni.com is structured as a system library designed to help remote workers build practical productivity frameworks. Instead of isolated articles, pages are organized into connected categories.

Readers can start with fundamental principles, then move toward planning systems, collaboration practices, and decision frameworks.

Recommended reading order

  1. Foundations — understand focus, deep work, and productivity constraints.
  2. Systems — build a weekly planning and daily review process.
  3. Team Work — learn async collaboration and documentation-first habits.
  4. Tools — choose tools intentionally instead of constantly switching.
  5. Comparisons — evaluate productivity setups based on role and context.

Who this site is for

This resource is designed for professionals working in distributed or remote-first environments, including freelancers, remote employees, and small teams.

  • Remote workers building personal productivity systems
  • Freelancers managing multiple clients
  • Startup teams working asynchronously
  • Managers coordinating distributed teams

Next steps

Begin with the foundational concepts and gradually build a system that matches your workflow and constraints.
